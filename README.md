# QR Code Generator

A simple web application that generates QR codes for the provided URLs.

## Features

- Input a URL and generate a QR code.
- Displays the generated QR code on the screen.

## Technologies Used

- HTML
- CSS
- JavaScript
- [QRious](https://github.com/neocotic/qrious) - QR code generation library

## Usage

1. Open `index.html` or 'jamesbrainard.github.io/qr-code-generator' in a web browser
2. Enter a valid URL in the input field.
3. Click the "Generate QR Code" button.
4. The generated QR code will be displayed on the screen and can be downloaded.
